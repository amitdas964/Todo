Welcome to Todo bash application (GUI).

A simple bash application where you can add/delete/view todo tasks. Useful for users who use terminal frequently.

Todo Features

  1. Add task
  2. View task
  3. Delete task
  4. On system login all the task list will be displayed first (GUI), so that user can remember and complete their tasks on time.

Future work

  1. Alarm feature
  2. Simple and user freindly android application
  3. Cloud storage and sync with android app
  4. Todo team feature, where you can create and manage teams and assign/manage tasks
